# Narrativ

AI-powered visual storytelling for social media. Research any topic, generate beautiful image carousels, and export ready-to-post content.

## Install

```bash
brew install chaiovercode/getnarrativ/narrativ
```

Requires macOS 12+ on Apple Silicon.

## Features

- **AI Research** - Deep research on any topic with automatic slide generation
- **Image Generation** - Create visuals with Gemini or Fal.ai
- **Custom Styles** - Extract and reuse visual styles from reference images
- **Local & Private** - Your data stays on your device, syncs via iCloud
- **No Subscriptions** - Bring your own API keys

## Setup

1. Install via Homebrew
2. Choose a vault location (iCloud recommended)
3. Add your API keys in Settings:
   - [Google AI](https://aistudio.google.com/apikey) (required)
   - [Fal.ai](https://fal.ai) (optional)
   - [Tavily](https://tavily.com) (optional)
