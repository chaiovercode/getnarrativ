Narrativ is an AI-powered macOS app that transforms any topic into beautiful, shareable image stories.

brew install chaiovercode/getnarrativ/narrativ

Requires macOS 12+ with Apple Silicon (M1/M2/M3) and Homebrew

Features

AI Research

Deep research on any topic with automatic fact extraction and smart slide generation.

Image Generation

Create stunning visuals with Gemini or Fal.ai. Multiple aspect ratios for stories, posts, and more.

Custom Styles

Extract visual styles from any reference image. Save and reuse your favorite aesthetics.

Notes & Wiki Links

Take notes with markdown support. Link directly to your research boards with [[wiki links]].

iCloud Sync

Your vault syncs automatically via iCloud. Access your work across all your devices.

Local & Private

Your API keys stay on your device. No accounts, no tracking, no subscriptions.

Getting Started

  1. 1
    Install with Homebrew Run the command above in Terminal
  2. 2
    Set Up Your Vault Choose where to store your research. We recommend iCloud for sync.
  3. 3
    Add API Keys Go to Settings and enter your API keys (see below)
  4. 4
    Start Creating Enter a topic and click Narrativ to generate your first story!
